# FloorPlanCAD Object Detection

Multimodal deep learning model for text-conditioned object detection in CAD floor plan drawings.

## Architecture

```
Image (PNG) в†’ VAE Encoder в†’ Image Tokens + 2D Positional Embedding

Built-in class texts (35 prompts):
  "Find {class} in this floor plan drawing"
        в†“
Text Encoder в†’ Text Tokens
        в†“
Per-class Early Fusion (FiLM / optional Cross-Attention)
        в†“
35 dedicated Object Learning pathways
(Mamba-like token mixer + Self-Attention + FFN)
        в†“
CenterNet Head [5 channels]
  в”њв”Ђв”Ђ center_heatmap [1, H/8, W/8]
  в”њв”Ђв”Ђ size_map       [2, H/8, W/8]
  в””в”Ђв”Ђ offset_map     [2, H/8, W/8]
```

## Dataset

**FloorPlanCAD** вЂ” 15,663 CAD floor plan drawings, 35 object classes.

The active training pipeline reads directly from the original flat dataset. No image copying and no crop input are required.

```
data/FloorPlanCAD_original/
  train_set_1/
    abc.png
    abc.svg
    abc_meta.json   # generated by scripts/data/build_dataset.py
  train_set_2/
  test_set/
```

Each training sample is one `(image, class)` pair. The model input at inference remains **one image only**; all 35 class prompts are built into the model.

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate

# Install torch build matching your CUDA first, e.g.:
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu121
pip install -r requirements.txt
```

## Data Preparation

```bash
# 1. Download original dataset
python scripts/data/download_gdrive.py

# 2. Generate metadata next to original PNG files
python scripts/data/build_dataset.py --data_root ./data/FloorPlanCAD_original

# 3. Verify output-resolution CenterNet targets
python scripts/dev/test_dataset.py --data_root ./data/FloorPlanCAD_original
python scripts/dev/viz_heatmap.py --data_root ./data/FloorPlanCAD_original --split train --index 0
```

## Training

```bash
python train.py \
  --data_root ./data/FloorPlanCAD_original \
  --image_size 512 \
  --output_stride 8 \
  --model_dim 256 \
  --depth_per_class 2 \
  --fusion_mode film \
  --batch_size 8 \
  --epochs 50 \
  --lr 1e-5 \
  --focal_weight 10.0 \
  --size_weight 1.0 \
  --offset_weight 1.0 \
  --warmup_steps 500 \
  --sampler balanced \
  --balance_power 0.5
```
